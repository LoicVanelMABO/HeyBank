# Layout Module

Ce dossier contient les composants de mise en page de l'application.

## Structure

- **header/** : En-tête de l'application (navigation, logo, menu utilisateur)
- **footer/** : Pied de page
- **sidebar/** : Menu latéral de navigation

## Règles

- Les layouts sont réutilisés dans toutes les pages
- Utiliser des composants standalone
- Centraliser la logique de navigation
