# Shared Module

Ce dossier contient les composants, directives et pipes réutilisables dans toute l'application.

## Structure

- **components/** : Composants UI réutilisables (boutons, modals, cartes, etc.)
- **directives/** : Directives personnalisées
- **pipes/** : Pipes personnalisés (formatage date, devise, etc.)

## Règles

- Les éléments doivent être génériques et réutilisables
- Pas de logique métier spécifique
- Exporter tous les composants/directives/pipes via un module shared
